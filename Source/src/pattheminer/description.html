A benchmark for Pat The Miner

Pat The Miner is an extension to the BeepBeep event stream engine. It provides various functions and processors to perform the following things:

  1. Compute in real time whether a stream of events deviates from a reference "trend"
  2. Compute reference trends from an event stream or a collection of reference event streams
  3. Learn associations between windows of the same trace using a machine learning classifier

This benchmark measures the performance of these functionalities, using various types of input event streams, pattern computations, and distance metrics.

The results in this benchmark are mentioned in the following publication(s):

  • M. Roudjane, R. Khoury, D. Rebaïne, S. Hallé. (2019). Statistics and Data Mining for Event Stream Analytics. In Information Systems, Elsevier. Submitted.
  • M. Roudjane, R. Khoury, D. Rebaïne, S. Hallé. (2018). Real-Time Data Mining for Event Streams. In Proc. EDOC 2018, IEEE Computer Society, 123-134. DOI: 10.1109/EDOC.2018.00025. This paper uses an earlier version of this lab.
