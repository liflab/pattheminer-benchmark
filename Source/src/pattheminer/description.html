A benchmark for Pat The Miner

Pat The Miner is an extension to the BeepBeep event stream engine. It provides various functions and processors to perform the following things:

  1. Compute in real time whether a stream of events deviates from a reference "trend"
  2. Compute reference trends from an event stream or a collection of reference event streams

This benchmark measures the performance of these functionalities, using various types of input event streams, pattern computations, and distance metrics.

The results in this benchmark are mentioned in the following publication:

  • M. Roudjane, R. Khoury, D. Rebaïne, S. Hallé. (2019). Statistics and Data Mining for Event Stream Analytics. In Information Systems, Elsevier. Submitted.

This paper itself is an extended version of the following publication:

  • M. Roudjane, R. Khoury, D. Rebaïne, S. Hallé. (2018). Real-Time Data Mining for Event Streams. In Proc. EDOC 2018, IEEE Computer Society, 123-134. DOI: 10.1109/EDOC.2018.00025. This paper uses an earlier version of this lab, hosted on Zenodo.

Prerequisites

There are no prerequisites to run this lab. The self-contained JAR file is a bundle of all the dependent libraries:

  • BeepBeep
  • Pat The Miner
  • Apache Commons Math
  • WEKA

All the logs processed in the lab are either synthetically generated at runtime, or static files located inside the bundled JAR.
