A benchmark for Pat The Miner

Pat The Miner is an extension to the BeepBeep event stream engine. It provides various functions and processors to perform two things:

  1. Compute in real time whether a stream of events deviates from a reference "trend"
  2. Compute referece trends from an event stream or a collection of reference event streams

This benchmark measures the performance of these two functionalities, using various types of input event streams, pattern computations, and distance metrics.

The results in this benchmark are mentioned in the following publication(s):

  • M. Roudjane, R. Khoury, D. Rebaïne, S. Hallé. (2018). Computing Trends over Event Logs. In Proc. EDOC 2018, submitted.
